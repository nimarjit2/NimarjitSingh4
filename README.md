📊 Basic Financial Data Analysis using Python

As part of my financial analytics learning journey, I performed a basic statistical analysis on the balance sheet dataset of Ashok Leyland using Python in Google Colab.

 calculated the mean value of financial variables to understand the central tendency of the data.

🔎 Steps involved in the analysis:

Importing and loading balance sheet data from a CSV file

Cleaning the dataset by removing missing values

Converting financial columns into numeric format

Computing the average (mean) of financial values using NumPy

📈 Why this matters:
Basic statistical measures like the mean provide a quick overview of financial data and help identify patterns before moving into deeper analytics such as regression, machine learning, or predictive modeling.

💡 Key takeaway:
Even simple statistical analysis plays an important role in data preprocessing and exploratory data analysis (EDA) when working with financial datasets.
